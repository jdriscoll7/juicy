# Installation
## Dependencies
```bash
git clone https://github.com/jwd0023/juicy.git
python3 -m venv ~/venvs/juicy
source ~/venvs/juicy/bin/activate
pip3 install -r juicy/requirments.txt
```

## Verifying Installation
```bash
cd juicy
source ~/venvs/juicy/bin/activate
python3 -m juicy.test.accelerometer_test
```
